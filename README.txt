Control:
c: clean the canvas
f: change the color (Black, C, M, Y)
a, z: increase/decrease the quantity of the water

Effect:
1. 墨水有diffuse效果。
2. 正確的blending(顏料的削減型三原色)。
3. 畫筆隨移動會漸漸沒墨水。
4. wet和dry分開記錄，wet層會隨時間變淡，一段時間後會滲入dry層；dry層不受時間影響。最後結果為wet和dry兩層計算融合得出。
5. wet層會逐漸往周圍擴散，且在wet層上畫上新墨水會延長它變乾的時間，因此在wet層和dry層上畫新墨水會有不同的blending效果。
6. 可以調節水量，不同水量的墨水在wet狀態的效果不同。
7. 水量越多，乾越慢；水量越少，乾越快。
